# SplitKit

Old School RuneScape-inspired browser RPG. Free-to-play with optional Holder Mode perks.

## Overview

SplitKit is a pixel-style browser game where you can:
- Train skills (Woodcutting, Fishing, Cooking)
- Complete quests and explore multiple areas
- Play completely free without a wallet
- Optionally connect a wallet to unlock Holder Mode perks

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
git clone https://github.com/splitkitfun/SplitKit.git
cd SplitKit
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Free-to-play** core gameplay
- **Optional Holder Mode** with exclusive zones and cosmetics
- **Multiple skills** to train and level up
- **Quest system** with rewards
- **Inventory and banking** systems
- **Save slots** (extra slots for holders)
